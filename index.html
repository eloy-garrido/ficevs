Acerca del Sistema

Este sistema fue diseñado específicamente para profesionales de Medicina Tradicional China que necesitan una herramienta moderna, segura y fácil de usar para gestionar las fichas clínicas de sus pacientes.

Construido con tecnologías web modernas y Supabase como backend, garantiza la seguridad de los datos mediante Row Level Security (RLS) y permite el acceso desde cualquier dispositivo con conexión a internet.

El formulario multi-paso facilita la captura organizada de información, desde datos básicos del paciente hasta diagnóstico MTC completo, incluyendo lengua, pulso, síntomas y plan de tratamiento.
